About This Project

This tool helps identify Canadian stamps from 1851–2025 using visual pattern matching. Browse the catalogue, filter by decade, or search by keyword to find your stamp.

How to Use

  1. Browse by era — tap a decade to filter
  2. Search — type a year, topic, or keyword
  3. Compare visually — scan the grid for your stamp's design
  4. Get details — tap any stamp for its ID and info
  5. Research further — use the year, topic, and denomination to cross-reference in stamp catalogues for varieties

Features

  • Complete visual catalogue of Canadian stamps 1851–2025
  • Instant search by year, topic, colour, or keyword
  • Decade filtering for quick era browsing
  • Responsive design — works on phone, tablet, and desktop
  • Open source and community-driven

Contribute

Help expand this collection! This is an open source project and contributions are welcome.

How to Add Stamps

  1. Fork the repository on GitHub
  2. Add your stamp image to the appropriate decade folder
  3. Update data/stamps.json with the stamp details
  4. Submit a pull request

Image Guidelines

  • High quality scans (300+ DPI)
  • Square format preferred
  • Main stamp design only (not varieties)
  • File naming: 001-beaver-3d-1851.jpg

JSON Format

{
  "id": "001",
  "year": 1851,
  "mainTopic": "Beaver",
  "subTopic": "Nature & Wildlife: Animals",
  "denomination": "3d",
  "color": "red",
  "image": "images/1850s/001-beaver-3d-1851.jpg",
  "notes": "First Canadian stamp"
}
