About This Project

The Canadian Stamp Identifier is an open source visual identification tool covering 3,475 Canadian stamps from 1851 to 2026. Browse the grid, filter by decade, or search by keyword to find your stamp.

How to Use

  1. Browse by era — tap a decade pill to filter, tap again to show all
  2. Search — type a year, topic, colour, or keyword (searches notes too)
  3. Combine filters — search within a decade for precise results
  4. Compare visually — scan the grid for your stamp's design
  5. Get details — tap any stamp for ID, denomination, category, and historical notes
  6. Research further — use the year, topic, and denomination to cross-reference in official stamp catalogues for varieties and pricing

What's Included

  • 3,475 stamps across 175 years of Canadian postal history
  • Every stamp categorised — 15+ categories from Nature & Wildlife to History & Heritage
  • Historical notes on every stamp — context, series info, and notable varieties
  • Colour descriptions for every stamp
  • Responsive design — works on phone, tablet, and desktop
  • Works offline after first visit (service worker caching)

What's Not Included

This tool covers main stamp designs only. Varieties, errors, plate blocks, first day covers, and other specialist items are not catalogued — but notes may reference notable varieties where they exist.

About the IDs

The stamp IDs (#001, #002, etc.) are proprietary reference numbers created for this project. They are not Scott catalogue numbers.

À propos de ce projet

L'Identificateur de timbres canadiens est un outil d'identification visuelle à code source ouvert couvrant 3 473 timbres canadiens de 1851 à 2026. Parcourez la grille, filtrez par décennie ou recherchez par mot-clé pour trouver votre timbre.

Comment utiliser

  1. Parcourir par époque — touchez une pastille de décennie pour filtrer, touchez à nouveau pour tout afficher
  2. Rechercher — tapez une année, un sujet, une couleur ou un mot-clé (recherche aussi dans les notes)
  3. Combiner les filtres — recherchez dans une décennie pour des résultats précis
  4. Comparer visuellement — balayez la grille pour trouver le motif de votre timbre
  5. Obtenir les détails — touchez un timbre pour voir son no., sa valeur faciale, sa catégorie et ses notes historiques
  6. Approfondir — utilisez l'année, le sujet et la valeur faciale pour consulter les catalogues officiels pour les variétés et les prix

Ce qui est inclus

  • 3 473 timbres couvrant 175 ans d'histoire postale canadienne
  • Chaque timbre est catégorisé — plus de 15 catégories, de Nature et faune à Histoire et patrimoine
  • Notes historiques sur chaque timbre — contexte, séries et variétés notables
  • Descriptions de couleur pour chaque timbre
  • Design adaptatif — fonctionne sur téléphone, tablette et ordinateur
  • Fonctionne hors ligne après la première visite

Ce qui n'est pas inclus

Cet outil couvre les motifs principaux des timbres uniquement. Les variétés, erreurs, blocs de planche, plis premier jour et autres articles spécialisés ne sont pas catalogués — mais les notes peuvent mentionner les variétés notables lorsqu'elles existent.

À propos des numéros

Les numéros de timbres (#001, #002, etc.) sont des numéros de référence exclusifs créés pour ce projet. Ce ne sont pas des numéros de catalogue Scott.

Contribute

This is an open source, community-driven project. Every contribution — a single corrected note, a better colour description, or a missing image — makes the tool better for collectors everywhere.

Easy Contributions (No Code)

  • Spot an error? Open a GitHub Issue with the stamp ID and correction
  • Have a stamp image? Upload it in an issue — we'll handle the rest
  • Refine colours — many modern stamps are listed as "multicoloured" and could use more specific descriptions
  • Add context — know the story behind a stamp? Share it

Direct Contributions (Pull Request)

  1. Fork the repository on GitHub
  2. Add your stamp image to images/[decade]/
  3. Update data/stamps.json with the stamp details
  4. Submit a pull request

Image Guidelines

  • 300+ DPI scans, JPG preferred
  • Main stamp design only (not varieties)
  • Naming: [id]-[topic]-[denomination]-[year].jpg
  • Example: 001-beaver-3d-1851.jpg

Category Format

Use the Category: Subcategory format for subTopic. Examples: Nature & Wildlife: Animals, History & Heritage: War & Military, Arts & Culture: Music.

JSON Format

{
  "id": "001",
  "year": 1851,
  "mainTopic": "Beaver",
  "subTopic": "Nature & Wildlife: Animals",
  "denomination": "3d",
  "color": "red",
  "image": "images/1850s/001-beaver-3d-1851.jpg",
  "notes": "Canada's first stamp, depicting a beaver."
}

Questions? Open an issue — we're happy to help.
