About This Project

This tool helps identify Canadian stamps from 1851-2025 using visual pattern matching. Simply browse the timeline, zoom into the era that matches your stamp's style, and click to get the identification details.

How to Use

  1. Start zoomed out - see the entire timeline
  2. Navigate to your era - use visual cues or search
  3. Zoom in - see individual stamps clearly
  4. Click to identify - get ID number and details
  5. Research further - use the ID in stamp catalogs for varieties

Features

  • Visual timeline of Canadian stamps 1851-2025
  • Search by year, topic, or keyword
  • Zoomable interface for detailed viewing
  • Focus on main stamp designs (not varieties)
  • Open source and community-driven

Contribute

Help expand this collection! This is an open source project and we welcome contributions.

How to Add Stamps

  1. Fork the repository on GitHub
  2. Add your stamp image to the appropriate decade folder
  3. Update data/stamps.json with the stamp details
  4. Submit a pull request

Image Guidelines

  • High quality scans (300+ DPI)
  • Square format preferred
  • Main stamp design only (not varieties)
  • File naming: XXX-topic-year.jpg

JSON Format
